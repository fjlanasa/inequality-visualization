### Setting Up

Make sure you have [Node Package Manager](https://www.npmjs.com/) (npm) installed.

Once npm is set up:

```
git clone https://github.com/fjlanasa/inequality-visualization.git
cd inequality-visualization
npm install

npm start

open http://localhost:3000
```
